# Code Connect overview

Code Connect links your code components to Figma components so Dev Mode can show real, production examples

## What this skill uses
- Code Connect files in your repo
- `figma.connect` mappings for props and examples
- Optional publish step using the Figma CLI

## What this skill does not use
- The Code Connect CLI interactive setup
- The Code Connect UI for mapping
